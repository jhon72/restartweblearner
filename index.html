Хочешь стать веб-разработчиком? Не знаешь с чего начать?

Если вы только начали изучать все аспекты веб-разработки, несомненно уверен, что ввиду большого количества информации, забыть казалось бы простые вещи, вполне реально! На помощь приходит данный справочник. Тут вы узнаете, как настроить сервер, подготовить среду разработки, найдете все теги и их составляющие с подробным описанием. Научитесь делать сайты используя Bootstrap. НО! Это еще не все! Тут вы так-же сможете найти информацию о том, как сделать сайт более живым, используя JavaScript + познакомитесь с фреймворками. Научитесь работать с базами данных и освоите backend-разработку, написав свой первый сайт на PHP (+ фреймворк Yii2). Почему PHP? Потому что это самый популярный, востребованный и не сложный в обучении язык программирования! В заключении пробежимся по популярным CMS (система управления сайтом) и CRM, познакомитесь с хостингами и изучите их специфику работы.









Оглавление справочника

1) Установка, настройка Open Server под ваши проекты

3) Верстка Bootstrap (С использованием LESS и прочих плюшек)

5) Ознакомление с ассинхронным JavaScript + XML
6) Фреймворки для JS, такие как JQuery и Angular
7) Работа с СУБД MySQL
8) Работа с серверной частью. Пишем на PHP
9) Ознакомление с фреймворком для PHP - Yii2 Framework
10) Мир CMS
11) Мир CRM
12) Хостинги, с чем и как едят
